Day 3 – Operators and Expressions

Topic: Arithmetic, Relational, and Logical Operators

Content: Learned to perform calculations and comparisons in Java using:

  • Arithmetic Operators: +, -, *, /, %.
  • Relational Operators: ==, !=, >, <, >=, <=.
  • Logical Operators: && (AND), || (OR), ! (NOT).

Explored operator precedence, associativity, and evaluated complex expressions using parentheses.

Exercise: Wrote a program to check if a number is even or odd and if it is positive or negative.

Reflection: Understanding operators and expressions is crucial for making decisions and performing calculations in programs.
