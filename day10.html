Day 10 – Polymorphism

Topic: Flexibility in Code

Content: Explored polymorphism:

  • Compile-time (method overloading): same method name, different parameters.
  • Runtime (method overriding): child class redefines parent method.
  • Helps write flexible and reusable code.

Exercise: Created Shape class with area() method, extended for Circle and Rectangle classes.

Reflection: Polymorphism allows multiple behaviors with a unified interface, enhancing code flexibility.
