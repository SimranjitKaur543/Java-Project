Day 8 – Constructors and this Keyword

Topic: Object Initialization

Content: Learned how constructors work:

  • Default constructor: no parameters, initializes default values.
  • Parameterized constructor: sets values when object is created.
  • this keyword: refers to current object’s variables.

Exercise: Created a Book class with a parameterized constructor to set title and author.

Reflection: Constructors simplify object creation and initialization, reducing code repetition.
