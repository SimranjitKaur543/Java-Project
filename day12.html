Day 12 – Interfaces and Abstract Classes

Topic: Abstraction in Java

Content: Learned:

  • Abstract classes: cannot be instantiated, may have abstract methods.
  • Interfaces: blueprint for classes, all methods are abstract.
  • Implemented interfaces using implements keyword.
  • Provides structure without exposing implementation.

Exercise: Created Animal abstract class with sound() method, implemented in Dog and Cat classes.

Reflection: Abstraction helps design structured programs and separates implementation from behavior.
