ðŸ“‹ Project Overview

The Contact Book Application is a Java-based desktop application developed using Java and MySQL to store and manage personal contact information efficiently. The system allows users to save, view, update, and delete contact details such as Name, Mobile Number, and Email ID using a user-friendly interface built with Java Swing.

One of the key features of this project is data validation, where the application ensures that the same mobile number cannot be stored more than once. This helps in preventing duplicate entries and maintains data consistency in the database.

All contact information is stored in a MySQL database, managed through MySQL Workbench. Java Database Connectivity (JDBC) is used to connect the Java application with the database for performing CRUD (Create, Read, Update, Delete) operations.
