Day 4 – Control Statements

Topic: Decision Making and Loops

Content: Studied conditional statements and loops:

  • If-Else: Learned how to execute code based on conditions.
  • Switch: Used for selecting one option from multiple choices.
  • For, While, Do-While loops: Learned to repeat code blocks efficiently.
  • Break and Continue: Controlled loop execution.

Exercise: Wrote a program to print the multiplication table of a given number using a loop.

Reflection: Control statements are essential for guiding program logic and iteration.
