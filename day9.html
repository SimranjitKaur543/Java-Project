Day 9 – Inheritance

Topic: Reusing Code with Classes

Content: Learned about inheritance:

  • Single inheritance: child class inherits from one parent.
  • Multilevel inheritance: chain of inheritance.
  • Super keyword: access parent class methods/constructors.
  • Method overriding: redefining parent method in child class.

Exercise: Created Vehicle class and extended it with Car class, adding new features.

Reflection: Inheritance reduces code duplication and improves maintainability.
