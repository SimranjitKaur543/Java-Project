Day 2 – Java Basics

Topic: Syntax, Variables, and Data Types

Content: Learned the basic syntax of Java including:

  • Java is case-sensitive and statements end with a semicolon.
  • Comments: // single-line, /* multi-line */.
  • Identifiers and keywords: rules for naming variables and reserved words.
  • Primitive data types: int, float, double, char, boolean.

Practiced using the Scanner class for input and System.out.println() for output.

Exercise: Created a program to calculate the sum of two numbers entered by the user.

Reflection: I feel comfortable with basic syntax and input/output operations, which will help in writing more complex programs.
