Day 1 – Introduction to Java

Topic: Java Overview

Content: Java is a high-level, object-oriented, platform-independent programming language. Today I learned about Java's features such as:

  • Platform Independence: Java programs run on any system with JVM.
  • Object-Oriented: Supports classes, objects, inheritance, polymorphism, encapsulation.
  • Robust: Handles errors and memory management efficiently.
  • Secure: Provides features like bytecode verification and exception handling.
  • Multithreaded: Allows multiple threads to run simultaneously.

I also installed the JDK, set up environment variables, and explored the difference between JDK, JRE, and JVM.

Exercise: Verified Java installation by running java -version and javac -version in the terminal.

Reflection: Understanding Java architecture and environment setup gave me confidence to start programming efficiently.
