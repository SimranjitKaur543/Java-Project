Day 7 – Object-Oriented Programming Concepts

Topic: Classes and Objects

Content: Learned OOP fundamentals:

  • Classes and objects: defining attributes and behaviors.
  • Encapsulation: data hiding using private variables and getters/setters.
  • Abstraction: hiding complex implementation.
  • Polymorphism: ability to take multiple forms.
  • Inheritance: reusing existing code.

Exercise: Created a Student class with name, rollNumber, and displayInfo() method.

Reflection: OOP concepts provide structure, clarity, and reusability to programs.
