Day 13 – Exception Handling

Topic: Handling Runtime Errors

Content: Learned about exceptions:

  • Try-catch block: catch and handle exceptions.
  • Finally block: code executed regardless of exception.
  • Throw keyword: manually throw exceptions.
  • Custom exceptions: define user-specific error handling.

Exercise: Wrote a program to handle division by zero using try-catch.

Reflection: Exception handling ensures programs don’t crash and can manage unexpected errors gracefully.
